IF yes, then log the event and do nothing else - IF no, then alert/alarm. Basic ones are light-based - They require light, making them not very reliable.

Which of these is a TRUE about hybrid encryption?

Explanation

We use the security over an unsecure media from Asymmetric for the initial exchange and we use the speed and higher security of the Symmetric for the actual data transfer. The Asymmetric Encryption may send a new session key ever so often to ensure security.
